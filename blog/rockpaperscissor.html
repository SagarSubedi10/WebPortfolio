Rock Paper Scissors JavaScript Game



Rock, Paper, Scissors Game With An Avatar Twist

I created a game of rock paper scissors back when I was first learning JavaScript. This is a legacy project, not updated or maintain, although I am looking forward to going back and revamping it with all the new experience I have gathered. Upon pressing the start game button


Functions:

  • computerSelection()
    • will randomly select either rock, paper, or scissor to be used a the computers option
  • playerSelction()
    • will prompt the user to select either rock, paper, or scissor to be used as their option
  • playRound()
    • Will play a single round of the game taking two parameters, for both the player and computer options. Returns a string value that declares the winner of that round. This function is case insenstive so the user input has some flexibility
  • game()
    • the playRound() function will be used within this function and looped five times, keep score and pronouce a winner.
